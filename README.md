# spring-token-pattern

A reusable Spring Boot library that centralizes token generation, validation, propagation and integration with a dedicated token management API.

## Features

- Token generation
- Token validation
- Token propagation
- Token cache
- Token interceptor
- API integration
- Auto renewal
- JWT support

---

## Flow

Application
↓
Token Pattern
↓
Token API
↓
Generated Token
↓
Microservice Communication
